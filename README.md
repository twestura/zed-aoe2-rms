# zed-aoe2-rms

A [Zed](https://zed.dev/) extension providing syntax highlighting for [Age of Empires II](https://www.ageofempires.com/) [random map scripts](https://docs.google.com/document/d/1jnhZXoeL9mkRUJxcGlKnO98fIwFKStP_OBozpr0CHXo) (`.rms`).

Built on the [tree-sitter-aoe2-rms](https://github.com/twestura/tree-sitter-aoe2-rms) grammar.
